# 🛡️ AI Scam Detector

## 🚀 Day 21 of my 30 Days • 30 AI Websites Challenge

AI Scam Detector is an AI-inspired web application designed to help users identify potentially fraudulent messages before becoming victims of online scams.

The application analyzes suspicious text, estimates a Scam Risk Score, classifies the likely scam category, highlights suspicious indicators, and provides practical safety recommendations to increase cybersecurity awareness.

# ✨ Features

* Scam Risk Analysis
* Scam Score (0–100)
* Risk Level Detection
* Scam Type Classification
* AI-style Explanation
* Red Flag Identification
* Safety Recommendations
* Confidence Score
* Copy Analysis Report
* Reset & New Analysis
* Interactive Sample Messages
* Responsive Glassmorphism UI
* Mobile Friendly Design

---

# 📋 How It Works

1. Paste or type a suspicious message.
2. Click **Analyze with AI**.
3. The application evaluates the content.
4. View the Scam Score and Risk Level.
5. Review detected scam category.
6. Read the identified red flags.
7. Follow the recommended safety tips.
